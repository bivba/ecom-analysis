**Дашборд в metabase**
- Данные получены с помощью SQL из clickhouse
- Визуализация включает основные метрики для мониторинга "здоровья" интернет магазина:
  - *Денежные метрики*: Revenue, AOV
  - *Продуктовые метрики*: DAU, Retention Rate (когортный анализ по дням)
  - *Воронка продаж*: Конверсия из просмотра товара в корзину и из корзины в покупку
  - *Лучшие товары*: Топ товаров по популярности просмотра и по выручке
